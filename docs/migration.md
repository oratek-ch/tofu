# Documentation migration

## Overview

The TOFU documentation is no longer hosted in this GitHub repository.

All official documentation is now available on the Oratek website:

https://www.oratek.com/products/tofu

Each product page provides direct access to its dedicated documentation.

## Why this change

The documentation has been moved to:

- provide a more structured and consistent experience
- keep product information and documentation in a single place
- simplify maintenance and updates

## What this repository is for

This repository is now used for:

- reporting issues
- asking questions
- suggesting improvements
- sharing feedback

## Legacy content

Some historical documentation may still exist in previous commits of this repository.

If you need access to older versions, you can browse the repository history.

## Need help?

If you cannot find what you are looking for in the documentation:

- open an issue in this repository
- describe your problem or question

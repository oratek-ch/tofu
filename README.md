# TOFU by Oratek

This repository is the public support and issue-tracking space for the **TOFU family** by Oratek.

## Official documentation

The official documentation is now hosted on the Oratek website:

- [TOFU product page](https://www.oratek.com/products/tofu)
- [TOFU board documentation](https://www.oratek.com/documentation/tofu-board)
- [TOFU5 board documentation](https://www.oratek.com/documentation/tofu5-board)
- [TOFU5+ board documentation](https://www.oratek.com/documentation/tofu5plus-board)

## What this repository is for

Use this repository to:

- report issues
- ask technical questions
- suggest documentation improvements
- request clarifications
- share feedback about TOFU, TOFU5, and TOFU5+

## Before opening an issue

Please check the official documentation first.  
If your question is still unanswered or something is unclear, open an issue here.

When relevant, include:

- product name: TOFU / TOFU5 / TOFU5+
- hardware revision
- Compute Module version
- power supply used
- peripherals connected
- operating system / image used
- steps to reproduce the issue
- photos or logs, if available

## Need direct support?

For commercial, project, or integration requests, please contact Oratek through the website:

- [Oratek contact page](https://www.oratek.com/contact)

## About the TOFU family

The TOFU family is Oratek’s compact carrier board platform for Raspberry Pi Compute Modules, designed for development and industrial integration.
